# 📚 Java Backend Engineering Handbook

> A structured handbook to master Java Backend Development from First Principles.

---

## 🎯 Goal

The purpose of this repository is to build a deep understanding of Java Backend Engineering from scratch.

This handbook is designed to help me:

- Master Java from first principles.
- Understand the reasoning behind every language feature.
- Prepare for Java Backend interviews in product-based companies.
- Build production-ready Spring Boot applications.
- Maintain a long-term knowledge base for revision.

This repository focuses on **understanding**, not memorization.

# 📚 Study Philosophy

This handbook follows one rule:

> **Never memorize a concept before understanding why it exists.**

Every topic is studied using First Principles.

For every concept we answer:

1. Why was this introduced?
2. What problem does it solve?
3. How does it work internally?
4. When should it be used?
5. When should it NOT be used?
6. How is it used in real backend systems?
7. How do interviewers ask about it?
